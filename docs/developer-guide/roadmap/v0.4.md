# v0.4 — Public generator registry

> Status: not started. Requires v0.2 (local generators) to be stable first.

---

## Goal

Let the community publish and install generators without forking dot.
Like npm for generators, or Terraform's provider registry.

---

## What ships

### Publishing a generator

```bash
dot generator publish    # publish the current generator to the registry
```

Generators are published as versioned Go modules to `registry.dot.dev` (name TBD).

### Installing a community generator

```bash
dot generator install mathieu/dot-generator-fastapi
dot generator install mathieu/dot-generator-terraform@v1.2.0
```

Fetches the generator binary from the registry, verifies the checksum, registers it locally.

### Registry commands

```bash
dot generator search fastapi        # search the registry
dot generator info mathieu/fastapi  # show generator details
dot generator update                # update all installed generators
```

### Registry infrastructure

- Public index at `registry.dot.dev` (or GitHub-based, TBD)
- Generators are versioned and checksummed
- `dot generator install` pins the version in `.dot/generators.json`
